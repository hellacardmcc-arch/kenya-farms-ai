# Exact Preview

```
┌─────────────────────────────────────┐
│  Code + Styling + Data + Logic  →   │
│           EXACT Preview             │
└─────────────────────────────────────┘
```

## Structure

| Layer | Location |
|-------|----------|
| **Code** | `src/components/*.jsx` |
| **Styling** | `src/**/*.css` |
| **Data** | `src/data.js` |
| **Logic** | `src/logic.js` |

Edit any file → Vite HMR → **Exact preview** in browser.

## Run

```bash
npm install
npm run dev
```

http://localhost:3010
